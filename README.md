# TelemetryDataAlgorithm
A data algorithm that recieves telemetry data exported from observability platforms like Laminar or directly from the program like Skyvern, and conducts analysis on the data to find the root cause of why a specific action (that likely caused the program to fail) was performed. 

I've made a personal roadmap of 'tsk_524039714399071386', in which a Skyvern instance attempts to buy a Premium Individual Membership in a Dark Pattern-prone website.
